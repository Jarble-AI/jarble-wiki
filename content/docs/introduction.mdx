## What is Jarble?

Jarble is a **no-code AI bot deployment platform** that lets you deploy LLM-powered bots to messaging platforms without writing any code. Connect your bot to WhatsApp, Discord, Slack, Telegram, and more — all through a guided wizard.

## Key Features

- **Zero-code deployment** — Pick a runtime, configure an LLM provider, and deploy
- **Multi-channel** — One bot, multiple platforms (WhatsApp, Discord, Slack, Telegram)
- **Rich UI components** — Your bot can render charts, tables, forms, and interactive widgets
- **Component marketplace** — Discover and install community-built components
- **Web chat interface** — Every deployment gets a shareable web chat at `/d/[id]`

## How It Works

<Callout type="info" title="Three steps to deploy">
1. Choose a runtime (OpenClaw or ZeroClaw)
2. Connect your LLM provider (Anthropic, OpenAI, OpenRouter, etc.)
3. Deploy and connect your messaging channels
</Callout>

Each deployment runs in its own isolated container with persistent storage, network isolation, and automatic health monitoring.
